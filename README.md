# 💪 90-Minute Gym Workout Plan

A personalized, interactive 4-day gym program built for a working professional — designed to build **strength and fitness** simultaneously. The plan is delivered as a single, fully self-contained HTML file with no dependencies or internet connection required.

---

## 🎯 About This Project

This workout plan was built for someone who:
- Works a **Monday–Friday, 8 AM–5 PM** office job
- Weighs **98 kg** at **5'7"** and is targeting body recomposition
- Wants to train **4 days a week** for exactly **90 minutes per session**
- Wants to build **muscle mass** while improving **cardiovascular fitness**

---

## 📋 The Program

The plan follows a **Push / Pull / Legs / Full Body** split:

| Day       | Session            | Focus                              |
|-----------|--------------------|------------------------------------|
| Monday    | Push Day           | Chest, Shoulders, Triceps          |
| Tuesday   | Pull Day           | Back, Biceps, Rear Delts           |
| Wednesday | Active Rest        | Walk / Light activity              |
| Thursday  | Legs Day           | Quads, Hamstrings, Glutes, Calves  |
| Friday    | Active Rest        | Walk / Light activity              |
| Saturday  | Full Body + Cardio | Compound lifts + Conditioning      |
| Sunday    | Rest               | Full recovery                      |

---

## ⏱️ Session Structure

Every session is broken into timed phases totalling exactly **90 minutes**:

- **Warm-Up** — 10–12 minutes of activation and mobility
- **Main Lifts** — 54–60 minutes of strength training
- **Cardio Finisher** — 7–18 minutes (HIIT or steady state depending on the day)
- **Core Work** — included on Legs day
- **Cool-Down & Stretch** — 7–8 minutes of static stretching

A **visual timeline bar** in the app shows the breakdown of each session at a glance.

---

## 🏋️ Key Exercises

**Compound lifts (strength foundation):**
- Barbell Back Squat
- Conventional Deadlift
- Barbell Bench Press
- Barbell / Machine Row

**Accessory work:**
- Incline Dumbbell Press, Shoulder Press, Lateral Raises
- Lat Pulldown, Seated Cable Row, Face Pulls
- Romanian Deadlift, Leg Press, Walking Lunges
- Bicep Curls, Hammer Curls, Tricep Pushdowns

**Cardio & conditioning:**
- Rowing machine (steady state)
- Stationary bike / elliptical HIIT
- Incline treadmill walk
- Farmer's Carries, Battle Ropes (optional)

## 🖥️ Features

- **Interactive tabbed interface** — switch between workout days instantly
- **Visual session timeline** — colour-coded breakdown of each 90-minute session
- **Timestamped exercise tables** — know exactly when each phase starts and ends
- **Estimated time per exercise** — keeps you on track inside the gym
- **Nutrition tab** — daily macro targets and meal timing guide
- **Weekly schedule overview** — at-a-glance view of the full week
- **Fully offline** — single HTML file, zero dependencies, works anywhere

---

## 🚀 Usage

1. Download `workout-plan-90min.html`
2. Open it in any web browser (Chrome, Firefox, Safari, Edge)
3. Tap the day tabs to navigate between sessions
4. Take it to the gym on your phone or print it out

No installation, no login, no internet required.

## 📈 Progressive Overload Tip

To keep making progress, apply the **+2.5 kg rule**: once you complete all reps of a set comfortably with good form, add 2.5 kg the next session. Track your lifts in a notebook or app like **Strong** or **Hevy**.
